# Wazuh SIEM Lab

Лабораторная среда для изучения Wazuh SIEM и детектирования атак.

## Стенд

| Компонент | Роль |
|-----------|------|
| Wazuh 4.9  | SIEM-сервер |
| Windows 10 | Агент Wazuh, сбор логов |
| Kali Linux | Атакующая машина |
| VirtualBox | Среда виртуализации |

Все машины находятся в одной сети.

## Инструменты

`Wazuh 4.9` `VirtualBox` `Linux` `Windows`

## Разобранные атаки

Атаки находятся в папке [`attacks/`](attacks) - каждая в отдельной директории.
